操作キー設定の変更方法
OPSION画面の中のCONTROLSで、操作キー設定を変更できます。
ひとつのキーボードで２人まで対戦が可能です。また、パッドは４つまで接続可能です。
VIEW KEYS現在のキーボードのキー設定を確認することができます。
SET KEYS 1１Ｐのキーボードのキー設定ができます。
SET KEYS 2２Ｐのキーボードのキー設定ができます。
SET JOYSTICKゲームパッド／ジョイスティックのボタン設定ができます。
BACKOPTION画面に戻ります。
変更したい項目を選択するとキー変更画面になります。上から順に好みのキーを選んで押して下さい。
キーは上から順に押すごとに決定されます。
一番下にあるCAMERAのキーを決定するとキー設定が完了し、SETKEYS選択画面に戻ります。

* 全てのキーを決定せずにESCキーで戻ると、それまでに決定したキー設定は無効になります。
* 間違えたキーを入力した場合はESCキーで戻り、再度変更画面でキー設定をして下さい。
* キーボード設定の時、１Ｐと２Ｐで同じキーは選択できません。


Ｅｘｉｔメニュー
レース中にPAUSEボタンを押すと，ゲームを一時中断してポーズメニューを表示できます。
ディレクションキーでメニューを選択して決定して下さい。
CONTINUE:レースをそのまま再開します。
RETRY:レースをスタート時点からやり直します。
RETIRE:レースをリタイアしてSELECT MODE画面に戻ります。

※ご注意

設定を済ませておく必要があります。設定の方法については、Windows95のヘルプや、ジョイスティック/
ゲームパッドの取扱説明書をご参照ください。
