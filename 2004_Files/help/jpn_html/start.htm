ゲームの始め方
ゲームスタート
 『Ｓｏｎｉｃ Ｒ』のタイトル画面が表示されたらスタートボタンを押してモードセレクト画面に進みます。
デモプレイになったらスタートボタンでタイトル画面に戻れます。

ゲームを続きから始めるとき
 セーブデータがある場合は、OPTION画面の「LOAD/SAVE DATA」を選択、決定して下さい。
データにはグランプリモードで集めたカオスエメラルドの数と、使用可能な隠れキャラが表示されて
います。↑↓でデータを選択して決定しましょう。画面下のメニューは、←→で選択して決定します。

LORDデータをロードします。はじめてゲームをする時はどのデータでもＯＫです。
SAVEデータをセーブします。
NEW新しくゲームを始めるときに決定します。
