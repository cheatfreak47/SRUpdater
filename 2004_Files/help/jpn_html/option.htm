OPTION 〜オプション〜
オプション画面
TIMESタイムアタックモードのベストタイムが見られます。
LOAD/SAVE DATALOAD/SAVE DATA画面に進みます。ゲームの途中でデータのロード/セーブができます。
GRAPHICS RESOLUTION      ：レゾリューションを変更することができます。
INTERLACE        ：インターレスを変更することができます。
DRAW DISTANCE：背景のポリゴン表示数を変更することができます。
WINDOW            ：画面サイズを変更することができます。
TRACK SHADING：ゲームフィールドのシェーディングを変更できます。
SOUNDゲーム中のサウンド設定ができます。
CONTROLS操作ボタンの設定ができます。詳しくは操作方法をご覧下さい。
GAME DIFFICULTY：ゲーム難易度を変更することができます。
GHOST       ：タイムアタックモードに出現するゴーストをON/OFFできます。
WEATHER   ：天候を変更できます。
CATCH UP  ：レース中に後者が前者に追い付く速度を変更することができます。
GUIDE        ：メインルートの星表示をON/OFFできます。
MAP          ：MAP表示のON/OFFができます。
2P SPLIT    ：2P対戦時の画面の分割方法をタテ、ヨコに変更できます。
EXIT TO WINDOWSゲームを終了し、WINDOWS画面に戻ります。
BACKゲームセレクト画面に戻ります。
