MULTI PLAYER 〜 ２Ｐ〜４Ｐ対戦モード 〜
１Ｐ〜４Ｐでレースをして、誰が速いかを競う対戦レースモードです。２Ｐ対戦は上下または左右２分割の
画面になっています。また、３〜４Ｐ対戦は４分割画面になります。
JOIN GAME
 画面上でタイムカウントが始まります。プレイに参加する人は、カウントが0になるまでに決定ボタンを
押して下さい。
SELECT MODE
RACEコースをふつうに3周して、誰が先にゴールできるか競います。
GET 5 BALLOONS誰が一番早く5つのバルーンを割ることができるか競います。バルーンはコース上に5つ
以上セットされているので、見つけたら絶対に対戦相手より早く割ってしまいましょう。

SELECT CHARACTER
 グランプリモードで使えるようになったキャラをセレクトできます。
SELECT COURSE
 グランプリモードで走れるようになったコースをセレクトできます。
