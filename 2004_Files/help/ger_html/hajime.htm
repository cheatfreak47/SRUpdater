Hinweise zu dieser Sega PC CD-ROM

Sicherheitsvorkehrungen:
1. Bei manchen Personen kann es zu epileptischen Anfällen kommen, wenn sie bestimmten Lichtmustern oder blinkenden Lichtern ausgesetzt sind. Diese Personen können einen Anfall erleiden, wenn sie bestimmten Fernsehbildern ausgesetzt sind oder bestimmte Videospiele benutzen. Es können auch Personen davon betroffen sein, deren Krankengeschichte bislang keine Epilepsie aufweist und die nie zuvor epileptische Anfälle gehabt haben. Falls Sie oder ihre Familienmitglieder zu epileptischen Anfällen neigen, konsultieren Sie vor dem Spielen bitte Ihren Arzt. Sollten während des Spiels Symptome wie Schwindelgefühl, Sehstörungen, Augen- oder Muskelzuckungen, Bewußtseinsverlust, Desorientierung oder jede Art von unfreiwilligen Bewegungen oder Krämpfen, so beenden Sie bitte SOFORT das Spiel und konsultieren Sie Ihren Arzt.

2. Achten Sie bei der Benutzung dieser Software auf einen gut beleuchteten Raum und eine angemessene Entfernung zum Bildschirm. Legen Sie im Interesse Ihrer Gesundheit jede Stunde eine Pause von 10 bis 20 Minuten ein und verwenden Sie die Software nicht, wenn Sie müde oder übermüdet sind.

Beschädigen und beschmutzen Sie die CD nicht:
Achten Sie darauf, die CD nicht zu zerkratzen oder zu beschmutzen. Verbiegen Sie die CD nicht und verändern Sie nicht den Durchmesser des Mittelloches.

Richtige Aufbewahrung:
Legen Sie die CD zur Aufbewahrung in ihre ursprüngliche Hülle und halten Sie sie von Orten mit hoher Luftfeuchtigkeit und hohen Temperaturen fern.

Beschriften Sie die CD nicht:
Beschriften Sie die CD nicht und bringen Sie keine Aufkleber darauf an.

Vorsicht bei der Reinigung:
Sollte eine Reinigung der CD notwendig sein, verwenden Sie dazu ein weiches Tuch wie etwa zur Linsenreinigung und wischen Sie vorsichtig in einer geraden Linie von der Mitte zum Außenrand. Verwenden Sie keine lösungsmittel- oder benzinhaltige Reinigungsmittel.

* Diese SEGA PC CD-ROM ist für den Einsatz in PCs. Versuchen Sie nicht, sie auf einem normalen CD-Spieler abzuspielen, da dies zur Beschädigung von Lautsprechern oder Kopfhörern führen kann.
* Bitte lesen Sie alle Liesmich-Dateien auf der CD, da sich dort wichtige Informationen befinden können.

Automatisches Abspeichern:
Diese Software speichert Bestleistungen und eingegebene Daten auf der im Menüpunkt SETTINGS (EINSTELLUNGEN) angegebenen Festplatte. Da dieses Abspeichern automatisch erfolgt, kann es zu Datenverlusten kommen, falls der Rechner ausgeschaltet wird, während das Spiel noch läuft.
