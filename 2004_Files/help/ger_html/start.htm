Spielbeginn
Nachdem die Logos von Sega und Traveller's Tales angezeigt wurden, erscheint das Titelbild von Sonic R. Drücken Sie START / RETURN, um den Bildschirm zur Wahl des Spielmodus aufzurufen oder warten Sie ein wenig, bis die Demo beginnt. Dort können Sie Sonic und seinen Freunden bei ihrem Rennen zuschauen. Um die Demo abzubrechen, drücken Sie START / RETURN.

Ein Spiel fortsetzen
Wenn Sie einen Spielstand gespeichert haben, können Sie ihn im Optionsbildschirm unter LADE/SPEICHERE DATEN laden.
Für jeden Spielstand wird die Anzahl der gefundenen Chaos-Edelsteine und die verügbaren versteckten Figuren angezeigt.

LADEN Lädt einen Spielstand.
SPEICHERN Speichert einen Spielstand.
NEU Beginnt ein neues Spiel.
