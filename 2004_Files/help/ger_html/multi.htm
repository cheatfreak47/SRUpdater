Multi Player (Mehrspieler-Modi)
In diesen Modi können bis zu vier Spieler gegeneinander antreten. Je nach Anzahl der Teilnehmer wird der Bildschirm entsprechend aufgeteilt.
Spiel beitreten
 Auf dem Bildschirm beginnt ein Countdown.
Die Teilnehmer müssen eine Taste drücken, bevor der Zähler 0 erreicht.
Wähle Spielmodus
RENNEN Ein normales Rennen über drei Runden.
FINDE 5 BALLONS Auf der Strecke sind Ballons versteckt. Sammeln Sie so schnell wie möglich fünf davon ein.

Wähle Spielfiguren
 Hier können auch Figuren gewählt werden, die im Grand Prix-Modus erspielt wurden.
Wähle eine Strecke
 Hier können auch Strecken gewählt werden, die im Grand Prix-Modus erspielt wurden.
