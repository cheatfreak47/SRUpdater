Time Attack (Zeitfahren)
Beim Zeitfahren können Sie aus vier Spielmodi wählen. Das jeweilige Ziel ist einfach: Machen Sie sich für das nächste Grand Prix-Rennen mit den Strecken vertraut.
Wähle Modus
NORMAL Ein normales Rennen über drei Runden.
RICHTUNGSWECHSEL Fahren Sie die Strecken zur Abwechslung mar rückwärts. Eine echte Herausforderung, sobald Sie den normalen Modus beherrschen..
FINDE 5 BALLONS Auf der Strecke sind Ballons versteckt. Sammeln Sie so schnell wie möglich fünf davon ein.
FANGE 4 CHARAKTERE Die anderen Sonic-Figuren starten mit etwas Vorsprung. Nachdem Sie gestartet sind, müssen Sie die anderen so schnell wie möglich fangen.

Nach dem Zeitfahren werden die Rennergebnisse angezeigt. Wählen Sie WIEDERHOLEN um die Strecke erneut zu fahren oder AUSGANG, um in die Modusauswahl zu gelangen.

Wähle eine Spielfigur
 Hier können auch Figuren gewählt werden, die im Grand Prix-Modus erspielt wurden.
Wähle eine Strecke
 Hier können auch Strecken gewählt werden, die im Grand Prix-Modus erspielt wurden.
