Screen Information

  Grand Prix

    
    1: Rings
    2: Rankings
    3: Sonic Tokens
    4: Total Time
    5: Lap Times
    6: Course Map


  Time Attack (Normal & Reverse Mode)

    
    1: Ghost
    2: Total Time
    3: Lap Times
    4: Course Map


  Time Attack (Get 5 Balloons Mode)

    
    1: How many balloons your character has broken so far.
    2: Total Time
    3: Course Map


  Time Attack (Tag 4 Character Mode)

    
    1: Runaway Character. Whan a character has been captured, their picture appears larger.
    2: Total Time
    3: Course Map


  Pausing the Game

    
    CONTINUE: Resume play.
    RETRY: Start from the beginning of the race.
    RETIRE: Return to the Select Mode screen.
