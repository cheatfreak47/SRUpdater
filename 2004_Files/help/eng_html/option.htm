Options

OPTION SCREEN

TIMES You can see your best time of Time Attack Mode.
LOAD/SAVE DATA You will go to the LOAD/SAVE DATA screen.You can load/save the data during a race.
GRAPHICS RESOLUTION :You can change the resolution.
INTERLACE :You can change the interlace.
DRAW DISTANCE :You can change the polygon display capability in the background.
WINDOW :You can change the screen size.
TRACK SHADING :You can change the game field shading.
SOUND You can change the sound options in the game.
CONTROLS You can change the key and button settings. Please refer to the Control.
GAME DIFFICULTY :You can change the gameplay difficulty level.
GHOST :You can turn ON/OFF the ghosts which appear on Time Attach Mode.
WEATHER :You can change the weather.
CATCH UP :You can change the speed to catch up with the character ahead.
GUIDE :You can turn ON/OFF the display of the Star on the main route.
MAP :You can turn ON/OFF the display of the Course Map.
2P SPLIT :You can select the screen dividing system for 2P VS mode, vertical or horizontal.
EXIT TO WINDOWS You will quit the game, and return to Windows.
BACK You will return to the Game Select screen.
