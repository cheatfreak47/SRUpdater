Instruction Manual for changing key setting
You can change key setting for game on the "CONTROLS" screen. ("CONTROLS" option is under "OPTION" screen)
2 Players can play on keyboard. Up to 4 game pads can be connected at once.

VIEW KEYS Change/Confirm Key setting on key board for game play
SET KEYS 1 Change/Confirm Key setting for player 1
SET KEYS 2 Change/Confirm Key setting for player2
SET JOYSTICK Change/Confirm Key setting for game pad/joystick
BACK Back to OPTION screen

Go to option (VIEW KEYS/SET KEYS/SET KEYS2/JOY STICK) if you wish to change the setting.
You can Confirm/Change Key setting listed up to bottom.
After setting up for "CAMERA" Key, all the Key settings will be confirmed and go back to SET KEYS screen.
Attention
All the key setting will be cancelled if you press "Esc" key before setting up for CAMERA key.
When you wish to change key setting which being selected for once, press Esc key to go back to OPTION screen.
Player1 and player 2 cannot select the same keys.


Pausing the Game
At any time during a race, press Start to pause the game. Select CONTINUE to resume play,
RETRY to start from the beginning of the race, or RETIRE to return to the Select Mode screen.
Press UP or DOWN to highlight the options, and press Start to select.

Notice
* When using a joystick or game pad, selection and confirmation is necessary before game play.
Open the [Game Controller] file in the [Control Panel] to select and confirm your game device.

* Game devices with four or more buttons are needed in order to play this game.
Game play is not guaranteed for devices having less than this number of buttons.
* This game is not designed for use with the Sega Saturn Control Pad, Virtua Stick or Arcade Racer.
Game performance using these devices cannot be guaranteed.
