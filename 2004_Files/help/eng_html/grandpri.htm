Grandprix Mode

Select Character
Sonic's quick, but all the characters have a chance at winning the race if
you time your turns correctly. Select as you did for the previous two
screens. Besides Sonic, you can enter the race as Tails, Knuckles, Amy,
maybe even Dr. Robotnik...more Xs? Hmm... I wonder what other hidden
characters there might be!

Select Course
Which course will you race? There are four (or maybe someday five, if you're good)
courses to choose from. Select the course as you did for the mode.
Each course is three laps in length. Hey, what's the X on the screen for?

Race Results
How did you do? On the Race Results screen, your final lap times and total time are listed.

Replay(for Grand Prix Mode)
Select this option to have the race you just ran displayed. Sit back and enjoy the show!

Retry(All Modes)
Care to try again? Race again with the same character and course.

Exit(All modes)
Select this to return to the Mode Select screen.
