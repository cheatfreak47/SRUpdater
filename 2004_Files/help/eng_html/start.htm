Getting Started
After the Sega and Traveller's Tales logos, the Sonic R Title screen
appears. Press Start from the Title screen to bring up the Mode Select
screen, or wait a little while for the demo and watch the Sonic
characters dodge and turn in their attempts to win the race. From the
demo, press Start to return to the Title screen.

To Continue a Game
If you have the saved data, select the LOAD/ SAVE DATA on the Option screen.
In the data, the number of Chaos Emeralds and the hidden characters you can use will be displayed.
Select the data with the OK Button.
You can also select the menu at the bottom of the screen with the OK Button.

LOAD You will load the data. When you first play a game, any data will be OK to use.
SAVE You will save the data.
NEW You will select to erase a previously saved game.
