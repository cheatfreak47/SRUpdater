Pour démarrer
Après les logos de Sega et de Traveller's Tales, l'écran titre de Sonic R apparaît. Appuyez sur Start pour afficher l'écran de Sélection de mode. Si vous attendez un peu, vous verrez la démo où les personnages de l'univers Sonic font de leur mieux pour gagner la course. Pendant la démo, appuyez sur Start pour revenir à l'écran titre.

Pour continuer une partie
Si vous avez des parties sauvegardées, sélectionnez CHARGER/SAUVEGARDER dans l'écran d'options.
Dans les parties sauvegardées, le nombre d'Emeraudes du Chaos ainsi que les personnages cachés que vous pouvez utiliser seront affichés.

CHARGER Vous chargez la partie. Quand vous jouez pour la première fois, tous les enregistrements seront disponibles.
SAUVEGARDER Vous sauvegardez votre partie.
NOUVEAU Vous choisissez de commencer une nouvelle partie.
