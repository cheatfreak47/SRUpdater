Conseils
Regardez le chemin emprunté par vos concurrents. Ils ont quelques fois des problèmes, mais la plupart du temps ils courent assez vite.
Sonic R est un jeu de course. Il n'échappe donc pas à la règle de ces jeux: entraînez-vous avec le mode Contre le chrono.
Explorez chaque circuit pour connaître l'emplacement des bonus.

    Sur chaque circuit, vous devez courir sur le chemin principal au moins une fois, mais n'oubliez pas les autres chemins.
    Ce qui peut apparaître à première vue comme une perte de temps vaut la peine d'être tenté.

    Lorsque vous jouez pour la première fois, concentrez-vous sur les bases de la course. Essayez les mouvements, habituez-vous aux circuits,
    et ensuite faites main basse sur les bonus!

    Le dérapage est efficace dans les virages serrés. Vous pouvez aussi freiner, mais dans les virages faciles, il vaut mieux
    se servir des touches directionnelles.
