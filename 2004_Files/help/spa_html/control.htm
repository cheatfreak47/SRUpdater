Para cambiar la configuración de las teclas
Puedes cambiar la configuración de las teclas seleccionando "CONTROLES" desde el menú de opciones
En el teclado pueden jugar dos jugadores, y hasta cuatro utilizando mandos de control.

Ver configuración Cambiar/Confirmar la configuración del teclado
Configuración del teclado 1 Cambiar/Confirmar la configuración del teclado para el jugador 1
Configuración del teclado 2 Cambiar/Confirmar la configuración del teclado para el jugador 2
Ajustar el Joystick Cambiar/Confirmar la configuración del mando de control
VOLVER Regresa a la pantalla de opciones


Atención
La nueva configuración será anulada si pulsas la tecla "Esc" antes de configurar la opción para CÁMARA.

Pausa
Durante el juego puedes pulsa el botón Start para hacer una pausa. Utiliza el botón direccional para seleccionar las diferentes opciones. Selecciona CONTINUAR para seguir con el juego, INTENTARLO DE NUEVO para volver a comenzar la carrera desde el principio, o RETIRARSE si quieres volver a la pantalla de selección de Modo.


Configuración de la pantalla
Durante el juego puedes utilizar las siguientes teclas para cambiar la configuración visual del juego:

F1 Disminuir la distancia de dibujo
F2 Aumentar la distancia de dibujo
F3 Mapa Activado/Desactivado
F4 Matiz de la pista Activado/Desactivado
F6 Polygon correction Activado/Desactivado
F7 Alpha-blending Activado/Desactivado
F9 Saltar imágenes Activado/Desactivado
F10 Entrelazado Activado/Desactivado (Sólo en alta resolución)
F11 Disminuir el tamaño de la pantalla
F12 Aumentar el tamaño de la pantalla



Nota
* Cuando utilices un joystick o un mando de control es necesario configurarlo previamente. Para configurarlo abre el programa "Dispositivos de Juegos" desde el panel de control de Windows.

* Necesitas un controlador de seis o más botones para jugar a Sonic R, si tu controlador no tiene esos botones, puede que el juego no funcione correctamente.
* Este juego no está diseñado para utilizar un Sega Saturn Control Pad, Virtua Stick o Arcade Racer. Si utilizas estos controladores, puede que el juego no funcione correctamente
